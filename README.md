# Entity Selection Outlines

Adds an outline when targeting certain entities including item frames, cushions, paintings, leash knots, armour stands, and end crystals.

## Planned features

There is a basic config system currently in the mod, however I want to make the entities the mod applies to much more customisable, possibly with a list of entity types to outline.

## Compatibility

This mod should be compatible with any mods or resourcepacks that change the colour of the block outline, including my own [custom outline creator](https://outlines.enchanted.games/) and coloured block outline packs on [vanilla tweaks](https://vanillatweaks.net/)
